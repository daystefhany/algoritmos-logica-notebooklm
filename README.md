# 📘 Lógica de Programação e Algoritmos com NotebookLM

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio da DIO, utilizando o NotebookLM como ferramenta de aprendizagem ativa.

O tema escolhido foi Lógica de Programação e Algoritmos, buscando compreender como o raciocínio lógico auxilia na resolução de problemas computacionais.

## 🎯 Objetivos

- Estudar lógica de programação.
- Compreender algoritmos.
- Desenvolver pensamento computacional.
- Utilizar IA como apoio aos estudos.

## 📚 Fontes Utilizadas

- Material da DIO.
- Apostilas de algoritmos.
- Conteúdo da disciplina de ADS.
- Documentação Java.

## 🤖 Prompts Utilizados

- Explique algoritmo para um estudante iniciante.
- Compare FOR, WHILE e DO WHILE.
- Como transformar problemas em algoritmos?

## 📖 Miniguia de Estudos

### Algoritmo
Sequência de passos para resolver um problema.

### Estruturas de decisão
IF e ELSE.

### Estruturas de repetição
FOR, WHILE e DO WHILE.

## 🚀 Conclusão

O NotebookLM auxiliou na organização do conhecimento e no aprendizado da lógica de programação.
